# Student Grade Calculator

## Project Overview
This Python project calculates student grades based on marks and displays encouraging messages.

## Features
- Accepts student name and marks
- Validates input (0–100 only)
- Uses functions and loops
- Handles errors safely

## Grading Logic
- A: 90–100
- B: 80–89
- C: 70–79
- D: 60–69
- F: Below 60

## How to Run
1. Install Python
2. Open terminal or command prompt
3. Run: python grade_calculator.py
